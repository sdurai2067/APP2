# PACKAGE PURPOSES:

#                Handles all route requests from browser
#
#   requests   : HTTP client library
#                Used to POST prompts to Gemini API
#                All Gemini calls go through requests.post()
#
#   python-docx: Read Microsoft Word .docx files
#                Extracts paragraph text from uploaded DOCX FSDs
#
#   pypdf      : Read PDF files page by page
#                Extracts text from uploaded PDF FSDs
#                Skips first 2 pages (cover/author pages)
#
#   openpyxl   : Create and format Excel .xlsx files
#                Builds the 3-sheet test case workbook
#                No API call needed — pure Python Excel generation
# ================================================================
